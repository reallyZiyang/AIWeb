# 2048 网页游戏 - 需求规范

## 1. 项目概述

- **项目名称**: 2048 网页游戏
- **类型**: 单页 Web 游戏
- **核心功能**: 经典 2048 数字益智游戏，支持键盘/触摸操作
- **目标用户**: 休闲游戏玩家

## 2. UI/UX 规范

### 布局结构
- 居中显示的游戏面板
- 最大宽度: 500px
- 整体居中 (flexbox)
- 顶部: 标题 + 分数 + 最高分
- 中间: 4x4 游戏网格
- 底部: 重开按钮 + 排行榜

### 视觉设计

**颜色方案**
- 背景: #0f0f23 (深蓝黑)
- 游戏面板背景: #1e1e3f (深紫蓝)
- 网格背景: #2a2a5a (中紫蓝)
- 卡片颜色:
  - 2: #eee4da
  - 4: #ede0c8
  - 8: #f2b179
  - 16: #f59563
  - 32: #f67c5f
  - 64: #f65e3b
  - 128: #edcf72
  - 256: #edcc61
  - 512: #edc850
  - 1024: #edc53f
  - 2048: #edc22e
- 文字色: #ffffff (深色卡片用白色，浅色卡片用深色)
- 按钮主色: #e94560 (珊瑚红)
- 按钮悬停: #ff6b8a

**字体**
- 主字体: 'Segoe UI', 'Microsoft YaHei', sans-serif
- 数字字体: 'Arial Black', sans-serif

**间距**
- 卡片内边距: 15px
- 网格间距: 10px
- 圆角: 8px

### 组件

1. **标题** - "2048"
2. **分数显示区**
   - 当前分数 (实时更新)
   - 历史最高分 (从本地存储读取)
3. **4x4 游戏网格**
   - 16 个格子
   - 数字方块动画 (出现/合并时)
4. **重开按钮** - "新游戏"
5. **排行榜** - 显示 Top 5 本地最高分

### 交互
- 键盘方向键控制 (↑↓←→)
- 触摸滑动支持 (移动端)
- 方块出现: 缩放动画
- 方块合并: 弹跳动画
- 分数变化: 数字跳动
- 游戏结束: 弹窗提示

## 3. 功能规范

### 核心功能

1. **游戏初始化**
   - 生成两个随机数字块 (2 或 4)
   - 初始化分数为 0
   - 加载本地存储的最高分

2. **移动逻辑**
   - 四个方向移动
   - 相同数字合并 (2+2=4, 4+4=8...)
   - 每次移动后生成一个新块
   - 移动后检测游戏结束

3. **分数系统**
   - 每次合并增加分数
   - 实时显示当前分数
   - 超过历史最高分时更新

4. **新游戏**
   - 重置网格和分数
   - 重新生成初始块

5. **本地排行榜**
   - 使用 localStorage 存储
   - 记录 Top 5 最高分
   - 显示排名、分数、日期
   - 新分数入榜时更新

6. **游戏结束检测**
   - 网格满且无法合并
   - 显示 "游戏结束" 弹窗
   - 显示最终分数
   - 检查是否进入排行榜

### 边界处理
- 键盘防抖 (快速按键不乱)
- 移动端触摸方向判断
- 分数溢出处理 (极大数字用 K/M 表示)
- localStorage 不可用时优雅降级

## 4. 技术实现

- 单 HTML 文件 (内联 CSS + JS)
- 无外部依赖
- ES6+ 语法
- CSS Grid 布局
- CSS 动画 (transition + keyframes)
- localStorage 持久化
