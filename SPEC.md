# 随机数生成器网页 - 项目规范

## 1. 项目概述

- **项目名称**: Random Number Generator
- **类型**: 单页 Web 应用
- **核心功能**: 生成指定范围内的随机整数
- **目标用户**: 需要生成随机数的用户

## 2. UI/UX 规范

### 布局结构
- 居中显示的卡片式布局
- 最大宽度: 480px
- 整体居中 (flexbox)

### 视觉设计

**颜色方案**
- 背景: #1a1a2e (深蓝黑)
- 卡片背景: #16213e (深蓝)
- 主色调: #e94560 (珊瑚红)
- 辅助色: #0f3460 (深蓝)
- 文字色: #ffffff (白色)
- 次要文字: #a0a0a0 (灰色)

**字体**
- 主字体: 'Segoe UI', sans-serif
- 数字字体: 'Courier New', monospace

**间距**
- 卡片内边距: 40px
- 元素间距: 20px
- 圆角: 16px

### 组件

1. **标题** - "随机数生成器"
2. **输入框组**
   - 最小值输入框
   - 最大值输入框
3. **生成按钮** - 主色调背景
4. **结果显示区** - 大号数字显示

### 交互
- 按钮悬停: 亮度提升
- 结果显示: 淡入动画

## 3. 功能规范

### 核心功能
- 输入最小值和最大值
- 点击按钮生成范围内的随机整数
- 显示结果

### 边界处理
- 如果最小值 > 最大值，交换两者
- 默认范围: 1 - 100

## 4. 验收标准

- [ ] 页面正确显示所有元素
- [ ] 可以输入最小值和最大值
- [ ] 点击按钮生成随机数
- [ ] 结果正确显示
- [ ] 样式符合规范
